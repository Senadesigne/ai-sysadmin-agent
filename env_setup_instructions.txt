INSTRUKCIJE ZA POSTAVLJANJE .env FAJLA
=====================================

KORAK 1: Kopirajte template fajl
--------------------------------
Kopirajte .env.example u .env fajl:

Windows PowerShell:
Copy-Item .env.example .env

macOS/Linux:
cp .env.example .env

KORAK 2: Generirajte CHAINLIT_AUTH_SECRET
-----------------------------------------
Generirajte svoj vlastiti auth secret:

python -c "import secrets; print(secrets.token_urlsafe(32))"

Zamijenite <GENERATE_ME> u .env fajlu sa generiranim secret-om.

KORAK 3: Dodajte API ključeve (opcionalno)
-------------------------------------------
Uredite .env fajl i dodajte potrebne vrijednosti:

GOOGLE_API_KEY=your-google-api-key-here
SSH_KEY_PATH=/path/to/your/ssh/key

NAPOMENE:
1. NIKAD ne commitajte .env fajl u git
2. Za Google API key idite na: `https://makersuite.google.com/app/apikey`
3. Bez Google API key-a, RAG funkcionalnost neće raditi ali chat će
4. SSH_KEY_PATH je potreban za remote konekcije
5. Koristite .env.example kao template za sve potrebne varijable

NAKON POSTAVLJANJA .env FAJLA:
- Restartajte aplikaciju
- Sidebar bi trebao raditi
- Chat će raditi (možda bez RAG konteksta ako nema Google API key)
